# React + TypeScript Crash Course

This repository is based on a crash course that helps developers learn how to use TypeScript in React applications and move from JavaScript to type-safe code.

It focuses on practical examples and real-world patterns used in modern frontend development.

---

## 🎯 Purpose

The goal of this repo is to understand how to:

- Use TypeScript in React projects
- Improve code safety and scalability
- Work with components, hooks, and global state
- Refactor JavaScript code into TypeScript

---

## 🧠 Topics Covered

### ⚙️ Project Setup
- Creating a React app using Vite with TypeScript
- Basic configuration

### 🧩 Props Typing
- Using interfaces to define props
- Ensuring strict component structure

### 🎣 React Hooks
- Typing useState correctly
- Handling events like onChange and onSubmit
- Using React event types

### 🌐 Context API
- Adding TypeScript to Context API
- Typing global state and providers

### 🔢 Enums
- Defining fixed sets of values
- Improving code consistency

### 🔄 JavaScript to TypeScript Refactor
- Converting JS components to TypeScript
- Fixing common type errors

---

## 🚀 Outcome

By working through this repo, you will learn how to:

- Build React applications with TypeScript
- Write safer and more maintainable code
- Apply real-world frontend patterns
- Understand how professional projects are structured
